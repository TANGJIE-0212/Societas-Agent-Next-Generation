# Societas Agent Next Generation

Standalone research package for Societas 3.0 agent direction.

## What is inside

- `index.html` — landing page for the research package.
- `self-evolving-agent-frontier-2026.html` — long-form frontier report on self-evolving agents.
- `self-evolving-agent-frontier-2026.md` — Markdown source for the long report.
- `next-generation-agent-directions-2026.html` — short strategic memo on next-generation agent directions and MVP path.

## Core thesis

Self-evolving agent is likely a core capability of next-generation agents, but not the whole picture. The near-term practical direction is:

```text
eval-first + computer-use + workflow + memory
```

The first MVP should be a verifiable execution agent. The second MVP should add lightweight self-evolution through trace logging, evaluation, memory extraction, skill cards, workflow checklists, tool policy, and human-gated updates.
